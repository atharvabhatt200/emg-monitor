About the project:

In this project, we aim to create a device to transmit raw Electromyography (EMG) signals to a remote server using IoT (Internet of Things) based technologies and detect various neuromuscular diseases using Digital Signal Processing and Machine Learning.

This project is a novel system that records and transmits electromyography (EMG) signals via WiFi. The device uses advanced signal processing techniques to extract the relevant features from the EMG signals and transmits the data wirelessly to a cloud-based server. The server then uses machine learning algorithms to perform fundamental analysis and prediction on these signals.

The system has the potential to revolutionize the way in which medical diagnoses are made. With its ability to remotely collect and analyze EMG signals, the system allows for a more comprehensive diagnosis of various neurological disorders, such as neuropathies, muscular dystrophies, and motor neuron diseases. The use of machine learning algorithms also enables the system to provide real-time monitoring and feedback to patients, enabling them to manage their conditions better.

While the system is primarily designed for basic analysis and prediction, it can also be used by qualified diagnosticians to perform more in-depth analysis and provide a more accurate diagnosis. Overall, this novel device has the potential to significantly improve the accuracy and efficiency of medical diagnosis for neurological disorders, leading to better patient outcomes and improved quality of life.
